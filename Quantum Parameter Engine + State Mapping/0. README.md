## The System in One Sentence

VQE runs offline on IBM Eagle quantum hardware to compute Fe₂O₃ corrosion parameters → those parameters feed a per-zone simulation → the simulation outputs typed state-transition deltas → Kanishk's deterministic engine consumes those deltas and makes maintenance decisions.
